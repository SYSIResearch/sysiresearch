Unlocking Embedded Innovations with RISC-V at RustTechX Summit

Dive into the forefront of embedded systems at RustTechX Summit with this insightful session on advancing RISC-V architecture using Rust. This talk sheds light on how Rust is shaping the future of low-level development and driving innovations in the open hardware movement. Perfect for enthusiasts of embedded systems, open ISAs, and those eager to explore how Rust powers real-world hardware solutions.

Building Reliable Systems with Rust at Scale

In this engaging talk, Abhishek Bhattacharjee shares real-world experiences of scaling production systems using Rust. The session focuses on how Rust’s powerful type system, memory safety, and performance features can be leveraged to build robust and maintainable backend infrastructure. Perfect for developers interested in building scalable services, improving reliability, and adopting Rust in real-world deployments.
